## 專案主題tea-shop

- 使用 Vue3 及狀態管理 Pinia 開發，具有電商購物車的功能
- 使用 mock API 來實現 axios 的 API 串接功能
- 使用 JWT 實現 Token-baseed 驗證登入/登出功能
- 導入i18n國際化，實現切換語系的翻譯功能
- 前端UI框架為 Element plus
- 樣式使用Tailwind及SCSS

### 專案安裝流程

1. 下載此專案到本機電腦

```sh
git clone https://github.com/tai0525/tea-shop.git
```

2. 進入此專案資料夾

```sh
cd tea-shop
```

3. 安裝 npm 套件

```sh
npm install
```

4. 啟動專案

```sh
npm run dev
```
